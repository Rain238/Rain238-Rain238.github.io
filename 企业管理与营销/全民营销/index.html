智能互联网时代企业营销的进化与突破，以坚持人格化营销为基本原则，以坚持企业超级大 IP 矩阵网为基本策略。

# 序章

未来提前到来，疫情给互联网带来的十大变革如下：

  1. 5G 技术的大规模商用加速推进。
  2. 人工智能、区块链、云计算、大数据、边缘计算、物联网数字技术 (简称： ABCDEI 数字技术) 进入大规模应用阶段。
  3. “数字鸿沟” 有望加速填平。
  4. 网络视频步入发展的黄金时期。
  5. 电子商务将迎来整体升级换代。
  6. 远程协同办公软件迎来新机遇。
  7. 医疗健康网络应用发展红火。
  8. 网络媒体格局变革将进一步深化。
  9. 互联网将进一步推动政府工作。
  10. 数字技术将创新社会治理方式。

# 竞争战略催生全民营销

  1. 优胜劣汰，适者生存。
  2. 竞争战略是全球所有经济体的生产法则。
  3. 智能互联网新时代的九大竞争态势。
    1. 人工智能 PK 人类技能。
    2. 线上平台 PK 线下平台。
    3. 主动优选 PK 被动选择。
    4. 深度内容 PK 广而告知。
    5. 全名网红 PK 大牌明星。
    6. 万物皆媒 PK 传统媒体。
    7. 娱乐购物 PK 计划购买。
    8. 知民品牌 PK 知名品牌。
    9. 全民营销 PK 全员营销。

# 企业应战新思维的人格化营销

  • 重构以后的新人货场模式有三个特点：
    • 一是以我 (人格) 为中心展开。
    • 二是关注初心原点 (开展人格化营销)。
    • 三是三大要素的对应关系发生改变，也就是谁喜欢我 (顾客)，我的价值是什么 (货)，我适合在哪里销售 (场)。
  • 人性的根本需求形成了 “人格化营销” 的基础。
  • 人人要有发言权、人人要有参与感、人人追求存在感、人人追求优越感。
  • 这是人的本性，这是全民营销以智能互联网营销战略为中心。
  • 以坚持人格化营销为基本原则。
  • 人格化营销的核心是品牌人格化，人格化营销的标签是 IP。
  • 什么是人格化营销？
    • 直白讲述为：赋予产品人的性格和情感特征。
    • 形成品牌特有的价值：态度、格调、从而进行沟通。
  • 品牌人格化就是赋予品牌人的情感，与人共鸣与人拉近关系，进而更有效地销售。
  • 创始人是企业 IP 重要的个性基础。
  • 全民营销的四大原则如下：
    1. 以智能互联网营销战略为中心。
    2. 坚持人格营销。
    3. 坚持以企业超级大 IP 矩阵网为基本策略。
    4. 团结和影响企业内外网民达成新时期的营销战略。

# 矩阵网的高度搭建

  1. 纵向矩阵。
  2. 横向矩阵。
  3. 横纵联合矩阵。

# 全民营销 = 平台算法 x 视频内容 x 社交裂变

  • 找到自己的绝对优势，才会被关注。
  • 可以有很多方向，关键在于我们要主动创造和发现。
  • 一个好名字能带来 500 倍的流量增长。
  • 从本质上来讲营销不是卖东西，而是卖客户心中的认同与感动。
  • 让更多人相信才是全民营销的精髓。
  • 标志 = 图腾。
  • 人格定位的价值感、独特感才会实际落地。
  • 六句箴言：
    1. 打击对手。
    2. 区隔同类。
    3. 引发关注。
    4. 增进感情。
    5. 关联产品。
    6. 专家背书。
  • 好故事的脚本 = 秒引发关注 x 秒反转 x 秒涨粉。

# 为什么要做私域流量

  1. 能变现。
  2. 抗风险。
