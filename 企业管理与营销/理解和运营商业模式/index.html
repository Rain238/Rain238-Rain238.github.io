# 商业模式

  1. 客户需求。
  2. 客户导向。

从源头上强调把握客户需求至关重要。

# 什么是商业模式？

  1. 在资源使用方面，强调利用一切可以利用的资源，而不见得只是利用所属所控的资源。
  2. 在商业目的实现上，强调实现独特的功能而不是盈利。
  3. 在盈利方式上并不排斥直接盈利，但更多的是间接盈利。

# 商业模式的特点

  1. 利用一切可以利用的资源。
    • 只有利用好一切可以利用的资源才可以让盈利最大化。
  2. 重要的是实现功能。
    • 商业模式最核心的就是你为客户实现了什么样的独特价值。
  3. 盈利方式更多的是间接化和多元化。
    • 在商业模式经营思维下直接盈利，但更多的是间接盈利和多元化盈利。

# 商业模式的定义

  1. 在资源使用上强调利用一切可以利用的资源，而不是强调利用自己所控的资源。
  2. 更好地为使用者实现某种功能或某种服务。
  3. 在商业目的实现上更多的是强调多元化盈利方式。

# 不同创新程度的商业模式

  • 第一种状态是企业只对内部的生产要素进行调整。
  • 第二种状态是企业结合外部资源客户情况一起创新，更加深入地解决客户需求。
  • 第三种状态是创新程度最为强烈就是解决一个市场空白点或创造了一个更为新颖的品类。
  • 第四种状态是前瞻性地看市场发现市场空白点需求，根据需求创造一个符合需求的产品或服务。

# 客户消费的四个阶段

  1. 定位阶段：明确产品定位和优势。
  2. 传播阶段：通过有效的渠道让客户知道我们的优势和定位。
  3. 销售阶段：确定价格渠道和终端布局。
  4. 售后阶段：对客户使用效果追踪。

# C2B 阶段

从粗糙的创意雏形，到产生一个相对完美，并且有一定商业价值的产品是需要一个流程的。

  • 市场趋势 / 客户需求预测
    • 第一步前瞻性地看市场，发现市场的空白点或者寻找客户需求，没有得到满足的地方。
    • 并不是每一个空白点都值得去做创新，因为创新也存在风险和成本。
    • 空白市场容量是否足够大，这也是需要考虑的重要因素。
  • 倾听真实想法
    • 如果想快速去抢占市场，规避目前行业中的强大对手，精准地了解真实情况，了解客户的真实需求是什么。
    • 所以想在这些市场领先者做得好的领域，来挑战他们那无疑是找死。
    • 领先者也存在薄弱的地方，这个就是后者可以进行大力创新的地方。
    • 商业进攻总是从竞争对手薄弱处下手，也是从用户没有得到满足的地方开始的。
    • 一旦到了成品阶段就要强化自己的特色大力宣传。
    • 收集有效意见。
    • 定位思路。
    • 社会化营销。

# 覆盖和渗透

  • 最为严重的情况就是企业原来的价值被覆盖。
  • 企业的核心利润受到了攻击，企业没有了存在的基础。
  • 信息不对称走向信息对称是完全不可逆的，所以在这一点上是连转型的机会都没有。
  • 新生代企业以颠覆的方法，去攻击旧生代去企业核心利润的时候，它能否成功的关键在于，能不能给原来的用户带来相当好的体验和效果。

# 丰富多彩的盈利方法

  • 需要强调的是商业模式思维，首先强调的并不是盈利而是解决用户需求的思想。
  • 重点是你要清晰地知道要解决什么问题，用什么方式合适地去解决这个问题。

往前拓展一步思路会更加开阔，在互联网时代需要新技术、新思维。
