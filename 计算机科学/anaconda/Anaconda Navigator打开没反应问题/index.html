Anaconda Navigator打开没反应问题

关于 Anaconda Navigator 在第一次安装完成后或用过 Anaconda Navigator 图形化页面，但在后续使用中突然打不开程序的问题。

# 解决方法

  1. 打开 Anaconda Navigator 文件所在位置，找到一个名为 Anaconda Prompt 的程序右键➡️以管理员运行。

  2. 升级 navigator ，执行 conda update anaconda-navigator

  3. 重置 navigator ，执行 anaconda-navigator --reset

  4. 升级客户端，执行 conda update anaconda-client

  5. 升级安装依赖包，执行 conda update -f anaconda-client

  6. 这样就可以重新启动 Anaconda Navigator 图形化页面了，到此问题解决。
