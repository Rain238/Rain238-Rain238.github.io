  • 自动增长通常都是和主键一起来食用

# 外键约束:foreign key

  • 让表与表产生关系，从而保证数据的正确性

# 非空约束:not null

# 唯一约束:unique

  • 某一列的值不能重复

  • 级联操作就是表与表关联后修改某一个的表的时候另一个表的数据会同步

# 事务回滚

  • 什么是事务回滚？
  • 事务回滚就是执行多条 SQL 语句的时候，要不同时成功，否则同时失败，从而保证了数据的安全性
