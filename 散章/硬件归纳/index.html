# 电阻欧姆公式

颜色单位 Ω
棕1Ω
红2Ω
橙3Ω
黄4Ω
绿5Ω
蓝6Ω
紫7Ω
灰8Ω
白9Ω
黑0Ω
金色误差 ± 5%(95%~105%)
银色误差 ± 10%(90%~110%)
无色误差 ± 20%(80%~120%)
  • 普通电阻计算公式是根据电阻上面的色环来计算出是多少 Ω 的电阻。
  • 四色环电阻 ：前两位代表正常值第三位代表需要在后面补几个 0 的个数，第四位则是代表误差正负值。
    • 如：橙红红，橙色代表 3 ，红色代表 2 ，拼在一起就是 32 但是第三位也是一个 2 ，也就是代表需要在末尾处补两个 0 ，最后就是 3200Ω ➡️ 3.2KΩ 。
    • 如：紫灰橙 ➡️ 78000Ω ➡️ 每三位进一则为千 78.000 ➡️ 78KΩ 。
    • 如：蓝绿黄 ➡️ 650000Ω ➡️ 每三位进一则为千 650.000 ➡️ 650KΩ ➡️ 每三位进一则为万 65MΩ 。
    • 如：灰红黑 ➡️ 第三位黑色代表 0 则不需要在后面补任何数，毕竟 0 代表 0 个数，➡️ 82Ω ➡️ 0.82KΩ 。
    • 例：▋▋▋ ➡️ 橙色 = 3 红色 = 2 黑色 = 0 ➡️ 320 ➡️ 32Ω ➡️ 0.32KΩ。
  • 五色环电阻 ： 五色环和四色环一样，只不过是五色环前三位代表正常值，第四位代表需要在后面补几个 0 的个数，第五位则是代表误差正负值。
    • 如：橙红红橙 ➡️ 322000 ➡️ 322KΩ ➡️ 32.2MΩ 。
    • 如：黄绿蓝红 ➡️ 45600 ➡️ 45.6KΩ 。
    • 如：黄绿蓝黑 ➡️ 456 ➡️ 0.456KΩ 。
    • 例：▋▋▋▋ ➡️ 橙色 = 4 绿色 = 5 蓝色 = 6 黑色 = 0 ➡️ 4560 ➡️ 456Ω ➡️ 0.456KΩ。
