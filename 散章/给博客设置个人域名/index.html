# 设置个人域名

  • 可以去阿里云 腾讯云 百度云 等购买一个域名
  • 可以注册一个 .cn .com .top 的看个人喜好注册
  • 我是在腾讯注册的 .top 的域名
  • 注册好域名后需要认证
  • 认证成功过后去添加解析
  • 来到解析页面选择添加解析
  • 现在主要需要添加的是这两条 @的解析记录
  • 下面框起来的是这两个解析记录的 SSL 证书这个到后面再说
  • 解析记录值必须填这两个 192.30.252.153 和 192.30.252.154 这是 GitHub 官方的 IP，不填的话绑定域名的的时候是无法来解析的

# 登录到 GitHub 仓库然后找到 Settings

  • 然后点进去找到 Pages

  1. 填入你的域名
  2. 域名填好点击 Save 添加，Remove 是删除
  3. 等上面出现 Your site is published at http://XXX.top/ 就好了，下面报红先不用管，等两分钟打开绑定好的域名看下是否有页面
  4. 有页面则就成功了，如果一直没有看下 DNS 解析是否配置正确
  5. 然后在你的博客文件 source 中创建一个名为 CNAME 文件，不要后缀，写上你的域名
